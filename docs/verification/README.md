# SDK Verification Artifacts

This directory stores lightweight verification outputs for SDK interoperability checks that are not part of the manuscript experiment folders.

## Tool Interoperability

`tool-interoperability-results.json` is generated by:

```bash
npm run verify:tool-interoperability -- --timeout-ms=900000 --probe-sigprofilerassignment-run
```

The verification checks SigProfiler-style and MuSiCal-compatible matrix round trips, SigProfilerAssignment matrix-mode execution in Chrome/Pyodide, SigProfilerExtractor and deconstructSigs handoff preparation, deconstructSigs and MuSiCal exposure import, the MuSiCal-compatible sparse NNLS refit path, and the combined multi-tool interoperability bundle.
