(async) plotCosineSimilarityHeatMap(groupedData, studyNameopt, genomeDataTypeopt, cancerTypeopt, divIDopt, conductDoubleClusteringopt, colorscaleopt, showTableopt) → {Promise.<Array.<Array.<number>>>}

Asynchronously plots a cosine similarity heatmap and optionally a corresponding table. This function takes grouped mutational spectra data, calculates the cosine similarity between samples, optionally performs double clustering to reorder the data, and then visualizes the similarity matrix as a heatmap using Plotly. It can also display a table showing the exact cosine similarity values alongside the heatmap.
Parameters:
Name Type Attributes Default Description
groupedData Object.<string, Object.<string, number>> An object representing the grouped mutational spectra data. The keys are sample names, and the values are objects where keys are mutation types and values are mutation counts. **Example:** ``` { 'Sample1': {'C>A': 10, 'C>G': 5, 'C>T': 12, ...}, 'Sample2': {'C>A': 8, 'C>G': 7, 'C>T': 15, ...}, ... } ```
studyName string <optional>
 "PCAWG" The name of the study. Used in the plot title.
genomeDataType string <optional>
 "WGS" The type of genome data (e.g., "WGS", "WES"). Used in the plot title.
cancerType string <optional>
 "Lung-AdenoCA" The type of cancer. Used in the plot title.
divID string <optional>
 "cosineSimilarityHeatMap" The ID of the HTML div element where the plot and table will be rendered. If the div does not exist it is generated.
conductDoubleClustering boolean <optional>
 true Whether to perform double clustering (on both rows and columns) to reorder the data in the heatmap and table.
colorscale string <optional>
 "RdBu" The Plotly colorscale to use for the heatmap. See https://plotly.com/javascript/colorscales/ for available options. **Examples:** "Viridis", "RdBu", "Blues", "Greens", "Hot".
showTable boolean <optional>
 false Whether to display a table showing the exact cosine similarity values alongside the heatmap.

Examples
// Example usage with default parameters:
const data = {
  'Sample1': {'C>A': 10, 'C>G': 5, 'C>T': 12},
  'Sample2': {'C>A': 8, 'C>G': 7, 'C>T': 15},
  'Sample3': {'C>A': 12, 'C>G': 4, 'C>T': 10}
};
plotCosineSimilarityHeatMap(data)
  .then(similarityMatrix => console.log(similarityMatrix))
  .catch(error => console.error(error));
// Example usage with custom parameters:
const data = {
  'SampleA': {'C>A': 20, 'C>G': 10, 'C>T': 5},
  'SampleB': {'C>A': 15, 'C>G': 12, 'C>T': 8},
  'SampleC': {'C>A': 18, 'C>G': 9, 'C>T': 6}
};
plotCosineSimilarityHeatMap(
  data,
  "MyStudy",
  "WES",
  "Breast-AdenoCA",
  "myHeatmapDiv",
  false,
  "Viridis",
  true
)
  .then(similarityMatrix => console.log(similarityMatrix))
  .catch(error => console.error(error));
