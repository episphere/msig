Local development

Run the same site from a clone

Reviewers can use the hosted notebooks immediately. Developers can clone the repository and run the local server to test unmerged changes.

  1. Clone the repository git clone https://github.com/episphere/msig.git
  2. Start the local static server npm run serve:observable
  3. Open the notebook gallery http://127.0.0.1:8080/notebooks/viewer.html
