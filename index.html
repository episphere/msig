Capabilities

From spectra to report

mSigSDK supports browser-based mutational-signature review from public or local inputs through validation, fitting, quality checks, visual review, and exportable reports. External tools remain separate engines; mSigSDK prepares handoff files and records provenance for those workflows.

Feature reference

Get data

Use mSigPortal and TCGA/GDC resources, local MAF files, and matrix imports.

Check inputs

Validate spectra, signatures, exposures, contexts, and MAF fields before analysis.

Run review

Fit known signatures, review panel/WES results, summarize cohorts, and run exploratory NMF.

Share outputs

Export QC figures, JSON/HTML reports, TSV files, handoff bundles, and provenance.
